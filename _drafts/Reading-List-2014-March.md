# 野火集

------------------

作者: 龙应台

《野火集》是龙应台在1984年在台湾的中国时报上刊登的专栏文章的集合，其中还包括了很多读者的评论。阅读30年前台湾的批评文章，发现其实任何社会都是存在问题的，重点是要有民主的机制，政府能够听得下批评的声音，勇于承认错误，才能真正将国家、社会治理好。
